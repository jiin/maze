## maze.pl
### Relaxing labyrinth generator

![Maze](http://www.uefap.com/images/maze.gif)

## Synopsis

```bash
perl maze.pl <width> <height>
```

## Description

maze.pl allows you to generate labyrinth for say 'NO' to the daily boredom.
